-----------------------------------------------
1. WBC1Cipher Class
   ✓ Key-dependent S-box generation using SHA-256
   ✓ Bit-level permutation tables (forward and inverse)
   ✓ Round key generation from master key
   ✓ Block encryption with multiple rounds:
     - S-box substitution (confusion)
     - Bit permutation (diffusion)
     - Cyclic byte rotation
